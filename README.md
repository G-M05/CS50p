# CS50p
Projects and exercises from Harvard CS50’s Introduction to Programming with Python.

This repository contains my coursework, exercises, and practice programs from Harvard's CS50P (CS50’s Introduction to Programming with Python).

I am currently using this course to strengthen my programming fundamentals, improve problem-solving skills, and build a stronger foundation in Python for future engineering and embedded systems projects.

Topics covered include:
- Variables and data types
- Functions and return values
- String manipulation
- Conditionals and loops
- File handling
- Data structures
- Problem solving and algorithmic thinking

I am documenting my progression throughout the course by organizing exercises by week and continuously improving my coding workflow using Git and GitHub.
